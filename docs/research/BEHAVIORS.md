# Seesaw Behavior Notes

Source URL: https://www.seesaw.website/

## Observed Interaction Model

- The page is primarily a static inspiration gallery.
- The top navigation is fixed at the top with a light border and white background.
- Desktop layout uses a fixed left category rail and a scrollable main gallery.
- Mobile layout removes the category rail and keeps the main content in a single column.
- Gallery cards are simple links with image thumbnails, compact metadata, and subtle hover affordance.
- Search and subscribe buttons are visible in the top navigation. Search opens an overlay on the source site, but the Zeron UI homepage keeps the search affordance as navigation to docs/search behavior.

## Responsive Notes

- Desktop at 1440px: fixed top bar, left rail around 250px wide, main content with three-column card grid.
- Mobile at 390px: no left rail, top content stacks, card grid becomes one column.
- The design depends more on spacing, density, and card rhythm than on heavy animation.

## Adaptation Notes

- The clone uses Zeron UI component-library content instead of Seesaw website entries.
- The gallery cards represent installable components, patterns, and blocks.
- The original site's visual pattern is retained: logo/search/CTA top bar, category rail, bold hero, and dense cards.
