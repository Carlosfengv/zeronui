## Copy-paste ownership

Zeron UI follows the shadcn/ui idea that component code should live in the product, where teams can inspect, adapt, and own it.

## Accessible primitives

Interactive primitives should prefer Base UI when a proven accessible behavior exists.

## Utility-first styling

Tailwind CSS is the styling layer. Tokens are still centralized so brand decisions are made once and consumed everywhere.
